LRCFS File Renamer

The LRCFS File Renamer is an application designed to automate the process of renaming large amount of files based on metadata.

With this application you can automatically rename hundreds of files from: DCS_0002.jpg to 20201231-Exp1-Rep2-UVLIGHT.jpg while simultaneously checking that data is valid for submission to our Transfer and Persistance projects.

Download the LRCFS File Renamer by clicking on the relevant link below for Windows and Mac. There are also instructions for use and an example data set to get you started. Please credit the Leverhulme Trust and LRCFS in publications where the LRCFS File Renamer is used.
