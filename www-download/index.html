LRCFS File Renamer

The LRCFS File Renamer is an application designed to automate the process of renaming large numbers of files based on metadata.

This application has been developed to help you automatically rename files based on a single metadata (.csv) file. This will enable submission of your data to the LRCFS Transfer and Persistance projects.

Download the LRCFS File Renamer by clicking on the relevant link below for Windows and Mac. There are also instructions for use and an example data set to get you started.
